# pppi
pr for pppi
"WEB APP" by "3 cowboys"
Це веб застосунок від ООО "3 ковбоя", призначений для швидкого виконання практичного завдання №3 з предмету "Професійна Практика Програмної Інженерії"
Інструкція запуску:
1. Відкрити класрум.
2. Відкрити класрум відповідного предмету.
3. Відкрити відповідне завдання відповідного класруму відповідного предмету.
4. Відкрити відповідний файл відповідного завдання відповідного класруму відповідного предмету.
Автори "3 cowboys":
1. Ілюша - Fronted + Database
2. Ростік - Backend + External Service
3. Жека - API Gateway + Infra
